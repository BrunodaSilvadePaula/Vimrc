  Vimrc
===

### Description

This is my .vimrc file

Yes, I use Vim as my main editor

### Dependencies

```sh
Vim-Plug
  https://github.com/junegunn/vim-plug
```

### Installation 

- Install dependecies
- Clone this project
- Open folder vimrc
- copy file .vimrc to the root machine

```sh
  $ sudo cp .vimrc ~
  $ sudo vim .vimrc
```

After that, run :PlugInstall from inside your newly made .vimrc
